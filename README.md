# 개요

해당 프로젝트는 토비의 클린스프링 강의를 보고 E 커머스 서버를 만들어보는 프로젝트입니다.  
bootstrap을 활용해 클라이언트를 제작하고 실제 유저 플로우가 동작하도록 개발합니다.  
포트원 테스트 API를 적용해 실제 환경과 유사하게 설계합니다.
